Mirrort

Mirrort is a puzzle platformer where
your goal is to reach the end of each
stage. But there is a twist, you control
two players at the same time and if one
dies, both go back to the start. So you
have to carefully make it to the other
side while none of the players fall.

This Project was created in my first
year of software development during a
project called "Shine" this was a big
demonstartion of all the things that
we had done that far. we invited
people to the demonstartion to come
see what I and others have created
