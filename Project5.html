The game I created for Project Play
is a Fighting game. The goal of the
game is to defeat your opponent. It
is a 2 player game designed to be
played on the arcade cabinet. The
one who first kills the other player wins.

I created this project together with
other Software Developers and individuals
specializing in Digital Design and Motion (DDM).
The goal of this project was to improve
collaboration with people. In the end, we
achieved what we set out to accomplish
and are pleased with the final product.
